# 海报网格识别与检索 — 使用逻辑与录屏口播脚本

## 一、页面结构（录屏时建议按这个顺序扫一眼）

| 区域 | 位置 | 作用 |
|------|------|------|
| 顶栏 | 最上 | 标题、说明、**中文 / English** 语言切换；另有链接可跳到「核心图像轮廓」子页面 |
| 左栏「海报分析」 | 可折叠 `details` | 上传海报、网格识别与微调、模板学习、入库、**版面区域分析** |
| 中间「画布」 | 主舞台 | 海报层 + 网格层 + 排版叠加层；支持拖拽区上传、缩放；底部 **状态行** 反馈操作结果 |
| 右栏「按网格查找海报」 | 固定 | 对已入库海报按列数、行数、分类、关键词 **检索** |
| 下方折叠面板 | 页面底部 | **颜色提取**、**主要图形与寓意**（含寓意词条检索） |

## 二、两条主使用路径

### 路径 A：从真实海报出发（识别网格）

1. **载入图片**：「选择海报」或把图片 **拖进画布区**，或 **Ctrl+V 粘贴**。
2. **自动识别**：载入后会在整张海报范围内 **自动识别网格**；不满意可点 **「重新识别网格」**。
3. **看参数**：`gridMeta` 区域显示列行、边距、模块尺寸、置信度等；可改 **列/行/边距/线型/颜色** 等，点 **「应用网格样式」**；需要与画幅比例一致时点 **「按画布贴合比例」**。
4. **确认网格**：点 **「确认当前网格」** 后，才可 **保存到网格库**（与模板学习等联动）。
5. **可选**：**学习当前网格** 存成模板，以后 **应用选中模板** 快速套网格。
6. **版面分析**：点 **「分析文字/图片区域」**，在版面上区分文字区/图片区并可叠加显示；分析后可进入排版编辑（见路径 B 的排版部分）。

### 路径 B：不依赖现成海报（空白画布排版）

1. 设 **画布宽高、背景色**，点 **「新建画布」**。
2. 从 **网格库** 选一个已保存的网格，点 **「应用网格库网格」**（或先上传海报走路径 A 把网格存入库）。
3. **添加文字块 / 图片块**，在画布上拖拽、缩放；可选 **自动排版** 生成多方案并在 **「AI排版方案」** 里切换。
4. **图层**：前移/后移、对齐画布、吸附网格、撤销/重做；可 **隐藏海报/网格** 便于纯画布编辑。
5. **导出**：排版图 PNG、网格 PNG、以及 **PSD / SVG / IDML** 等到设计软件里继续改。
6. **排版项目**：命名后 **保存当前排版 / 载入 / 删除**，与海报库分开管理。

### 路径 C：检索与复盘（右栏）

对已 **保存入库** 的海报，按 **列数、行数、分类、关键词** 筛选，卡片上可 **载入** 或 **删除**。

### 路径 D：底部分析（辅助）

上传海报后展开：**颜色提取**（主色、RGB/CMYK、复制 HEX）；**图形与寓意**（氛围、解读、词条检索）。用于讲解与灵感参考，不替代专业设计决策。

## 三、录屏口播脚本（约 3～6 分钟，可按需删减）

**【开场 · 15～25 秒】**  
大家好，这是一个在浏览器里运行的 **海报网格识别与检索** 工具。它能在整张海报上 **自动识别网格结构**，支持 **微调网格样式**、**把网格和海报归类入库**，还能做 **版面区域分析**、**空白画布排版**，以及 **导出到 PS、AI、InDesign**。页面顶部可以切换 **中文和英文** 界面。下面我们按界面从左到右、从上到下走一遍。

**【左栏 · 上传与网格 · 45～90 秒】**  
左侧是「海报分析」面板。第一步是 **上传海报**，支持 JPG 和 PNG；你也可以 **把图片拖进中间画布**，或者用 **Ctrl+V 粘贴**。上传后，系统会在 **全图范围** 内自动识别网格。  
中间这块是 **网格样式**：列数、行数、边距、倾角、颜色、虚线、线宽都可以改，改完点 **「应用网格样式」**。如果希望网格和当前画幅比例严格贴合，可以点 **「按画布贴合比例」**。  
下方 `gridMeta` 会显示识别结果的细节，比如列行、边距和置信度。  
当你对网格满意时，点 **「确认当前网格」**——这一步很重要，**确认之后**才能稳定地做 **保存到网格库** 和 **学习模板**。  
「模板学习」可以把当前网格配置存成名字，以后一键 **应用选中模板**。  
「归类入库」里填写海报名称和标签，把当前海报和网格 **存进网格库**，方便后面按网格检索。

**【版面分析 · 20～40 秒】**  
同一块面板下方有 **版面区域分析**。点击 **「分析文字/图片区域」**，系统会划分文字区和图片区，并可以选择是否 **叠加显示** 在画面上。分析完成后，通常会 **进入可编辑的排版模式**，方便后面调整图层。

**【左栏 · 排版 · 60～120 秒】**  
展开 **「海报排版」**。如果你 **没有海报**，可以设好 **画布宽高和背景色**，点 **「新建画布」**，再从 **网格库** 里选一个网格 **应用到画布**。  
也可以 **添加文字块、图片块**，用 **自动排版** 生成多个方案，在下拉里切换。  
中间画布上可以 **显示或隐藏网格和底图**，方便只看排版。选中元素后可以调 **字体、字号、对齐**，用 **图层前移、后移**，以及 **对齐到画布** 的六个方向。  
开启 **吸附网格** 时拖拽会贴齐网格线；需要微调时可以 **按住 Alt 临时关闭吸附**（以你界面提示为准）。  
排版满意后，可以 **导出排版图**、单独 **导出透明网格 PNG**，或导出 **PSD、SVG、IDML** 到专业软件。  
同一面板里还有 **排版项目的保存、载入和删除**，和海报库是分开管理的。

**【右栏 · 检索 · 25～40 秒】**  
右侧是 **按网格查找海报**。对已经入库的海报，可以按 **列数、行数、分类、关键词** 筛选，点 **查找**。每条结果可以 **载入回画布** 继续编辑，或 **删除** 记录。

**【底部 · 颜色与寓意 · 25～45 秒】**  
页面下方有两个可折叠模块：**颜色提取** 会从当前海报里提取主色，显示 RGB 和参考 CMYK，点击色块可复制 HEX。  
**主要图形与寓意** 会根据构图和颜色给一个 **启发式的氛围和解读**，下面还可以 **搜索寓意相关的关键词**。这部分适合作为参考，不是硬性设计规范。

**【收尾 · 15～25 秒】**  
总结一下：**上传或拖入海报 → 自动识别网格 → 微调并确认 → 入库或存模板**；需要精细排版时可以 **分析版面** 或 **新建画布** 做 **图层化排版** 并 **多格式导出**；右侧 **按网格检索** 历史海报；底部 **颜色与寓意** 辅助分析。感谢观看。

## 四、录屏实操顺序建议（分镜）

1. 顶部标题 + 语言切换（3 秒）  
2. 拖入一张海报 → 指一下自动出现的网格（8～15 秒）  
3. 改一两个网格参数 → 应用 → 指 `gridMeta`（15 秒）  
4. 确认网格 → 演示保存到网格库或学习模板（二选一，20 秒）  
5. 点一次版面分析（若时间紧可口述跳过实际操作）（15 秒）  
6. 切到「海报排版」：新建画布或继续当前图 → 加一个文字块 → 演示对齐或吸附（30 秒）  
7. 右栏输入条件检索（若有入库数据）（15 秒）  
8. 展开底部颜色 / 图形寓意（15 秒）  
9. 收尾一句话（5 秒）

## 五、口播时可用的「一句话定位」

- **这个工具解决什么问题**：把海报上的 **网格结构** 抽出来，变成可编辑、可检索、可复用的数据，并串起 **排版与导出**。  
- **和「核心图像轮廓」页面的关系**：顶栏链接是 **另一条线**（线段勾勒轮廓），本页核心是 **网格 + 排版 + 库**。
